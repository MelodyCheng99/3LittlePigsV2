## Instructions for Running the Application Locally

1. Open a terminal and navigate to the root of the application.

2. Enter the following command in the terminal: `cd server`

3. Enter the following command in the terminal: `flask run`

4. Open another terminal and navigate to the root of the application.

5. Enter the following command in the terminal: `cd client`

6. Enter the following command in the terminal: `npm start`
